Current projects

AI-Driven CBT Platform
LLM Agent

An agentic LLM-powered web application designed to modernize Cognitive Behavioral Therapy (CBT). By automating the documentation and structuring inherent in traditional protocols, the platform streamlines the therapeutic process through a sleek, intuitive UI. It offers 24/7 accessibility to an AI agent, providing a continuous complement to traditional clinical care.

Tourism Demand Analytics in Norway
LLM Agent Fullstack

An agentic LLM-powered web application that gives hotel investors and operators actionable insights into Norwegian tourist accommodation demand. Powered by public data from Statistics Norway (SSB) and the Brønnøysund Register Centre (Brreg), the platform lets users query the agent to generate visualizations and compare occupancy trends across regions and time periods.

Citybike Relancing Agent
LLM RAG

Toy project to test the scalability of an agentic framework for a logical planning task (rebalancing of citybikes). An agentic loop with multiple LLM and tool calls is used together with a RAG pipeline using Hopsworks feature store. The LLM (gpt-4o-mini) is called using in-context learning.
